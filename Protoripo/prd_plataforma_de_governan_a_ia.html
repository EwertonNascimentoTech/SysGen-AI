# Plataforma de Gestão e Governança de Projetos com IA - PRD ## 1. Visão Geral Centralizar gestão, rastreabilidade e documentação de projetos de IA (GitHub, Markdown, Cursor artifacts). ## 2. Entidades Principais - **Projeto:** Centro do sistema (Nome, PO, Diretoria, Prazos, Metodologia). - **Kanban:** Define o status operacional (Baseado em Templates). - **GitHub/Wiki:** Integração para documentação viva a partir da pasta /Doc. - **Cursor Hub:** Governança de Rules, Skills e MCP. ## 3. Perfis e Permissões - **Admin:** Controle total, usuários e auditoria. - **Coordenador:** Cria projetos e gerencia templates. - **PO:** Acompanhamento funcional e prazos. - **Dev:** Consulta técnica e evidências. - **Liderança:** View-only/Dashboard. ## 4. Fluxos Críticos - Cadastro de Projeto -> Seleção de Template -> Fluxo Kanban. - Vínculo GitHub -> Seleção de Tag -> Geração de Wiki (Read-only). - Governança de Artefatos Cursor (Rascunho -> Publicado).
